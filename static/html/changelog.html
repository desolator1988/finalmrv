Version 1.7

    Version 1.7
  • New layout based on Twitter Bootstrap

Version 1.6

    Version 1.6
    Project Design & Evaluation Tools
    July 2013
  • Added Plot Mapping Tool for generation locations for sample plots
  • Migrated to PostGIS
  • Added Project Emissions Tool for calculating non-CO2 emissions resulting from forest fires
  • Added Leakage Estimation Tool for calculating leakage emissions
  • Added Sampling Design Tool for determining the number of sample plots required for a project
  • Added Report Wizard for reporting projects using the GEF Tracking Tools
  • Added GEF Guidebook with pageslide access for quick reference by users
  • Embedded Guidebook button links in various locations for quick reference
  • Added default Allometric Equations to script for pre-populating database
  • Added basic permissions/sanity checking for modifying allometric equations
  • Minor UI improvements

Version 1.5

    Version 1.5
    Allometric Equation Editor
    March 2013
  • Added new feature for defining custom allometric equations
  • Added basic sanity-checking validators to ensure safe allometric equations
  • Created new Workspace Utilities section to contain AEQ editor, Add User form
  • Updated RESTful API back-end to always fetch 300 objects per page (from 20)
  • Fixed bug preventing deletion of practice/parcel/scenarios in e-calc
  • Slight tweaks of front-end Carbon Calculator decimal precision to improve readability

Version 1.4

    Version 1.4

  • Improved carbon stocks calculator to automatically ignore parcels/plots for which the data field is null. (Previously, user had to manually tick-box the data types.)
