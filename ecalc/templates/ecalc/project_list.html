Guide

The Project Wizard guides the user through the process of using the Carbon Calculator. Basic data must be entered to describe simple parameters for Project, Parcel, Land Cover, Reference Scenario, and Project Scenario.
Step 1:
The user describes the Project by entering data for Name, Duration, Continent, Climate, Moisture, and Soil Type.
This step is completed when a project is created in the MRV.

Step 2:
The user describes the Parcel by entering data for Name, Location, Area, and Land Cover. The user has an option of adding a new Land Cover with national Tier 2 values or their own Tier 3 data from field measurements if the Tier 1 IPCC default values do not accurately describe the Parcel’s actual Land Cover.

Step 3:
The user describes the Reference Scenario for the selected Parcel by entering data for the expected Land Cover, Degradation, Harvested Wood Products, Burning, and agricultural Practices. Users may add multiple land cover changes or multiple practices within a single Reference Scenario by modifying the scenario through the Project page.

Step 4:
The user describes the Project Scenario associated with the selected Reference Scenario. The Project Scenario also includes the expected Land Cover, Degradation, Harvested Wood Products, Burning, and agricultural Practices. Users may add multiple land cover changes or multiple practices within a single Project Scenario by modifying the scenario through the Project page.

Step 5:
The emissions calculations are reported on the Project page. The user must click on the Reference Scenario or Project Scenario emissions value to open the Emissions Table which reports annual carbon stocks and GHG emissions. The difference between the Reference Scenario and Project Scenario emissions values is also reported on the Project page.
Experienced users may choose not to use the Project Wizard and enter new project data independently using the “Add New” button on Project Details page.
